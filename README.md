# TruLight Georgia RepCard

Digital rep card and field sales platform for TruLight Georgia.

## Live Site
https://trulightgeorgia.com

## Features
- Digital business card with QR code sharing
- Lead tracker and pipeline management
- Interactive neighborhood canvassing map (Alpharetta metro)
- Team leaderboard
- Review request flow
- Auto follow-up settings

## Deployment
This site is hosted on GitHub Pages with a custom domain (trulightgeorgia.com).

## Tech Stack
- Vanilla HTML/CSS/JavaScript
- Leaflet.js for interactive maps
- Progressive Web App (installable on mobile)

## Install as App (Mobile)
1. Open trulightgeorgia.com in Safari (iOS) or Chrome (Android)
2. Tap Share → "Add to Home Screen"
3. The app installs with a TruLight icon
